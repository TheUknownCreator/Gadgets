# Gadgets
Light-weight tools

## RightMenuAddToPATH `[Win11 25H2 10.0.26200.8655]`
Add options for adding folder to PATH to the right click menu.
### Install
Put the "RightMenuAddToPATH_Installer.ps1" in the folder where you want to generate the script and run it as Administrator or double-click "RightMenuAddToPATH_Installer.cmd".

## FileExplorerRestarter `[Win11 25H2 10.0.26200.8655]`
Restart the `explorer.exe` and reopen all folders in their original windows and tabs.
